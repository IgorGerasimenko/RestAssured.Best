# RestAssured.Best
Rest assured best practices


### Всем привет! Это пример использования библиотеки RestAssured для автоматизации API тестов.

В тестах используем:

-Пагинацию

-Циклы

-Сериализацию

-Pojo классы

-Stream api

-Счетчик запусков

-Cервис для генерации POJO: https://json2csharp.com/json-to-pojo
